# 🌐 FSD Lab 6 — PWA Pokédex

A **Progressive Web App (PWA)** Pokédex built with React + Vite. Installable on any device and works offline.

## ✨ Features
- PWA support — installable as a native-like app
- Service Worker for offline caching
- Web App Manifest with icons
- Fetches Pokémon data from [PokéAPI](https://pokeapi.co/)
- Responsive layout with modern card design

## 🛠️ Tech Stack
- React 19
- Vite + vite-plugin-pwa
- Axios
- Vanilla CSS

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
